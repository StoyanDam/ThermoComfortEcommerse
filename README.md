ThermoComfort is a full-stack e-commerce platform dedicated to residential heating solutions, such as heat pumps, air conditioners, and pellet stoves. This project was developed as a graduation thesis for a professional qualification in "Applied Programming" at the High School of Mathematics "Konstantin Velichkov", Pazardzhik.

The application is built using the MVC (Model-View-Controller) architecture, ensuring high performance, scalability, and a clean separation of concerns.

## 🚀 Key Features

### User Capabilities
* **Product Catalog:** Detailed browsing of products with specific technical characteristics and availability.
* **Shopping Cart:** Add/remove products and dynamic total price calculation.
* **Order System:** Integrated checkout process with an automated logic that calculates the delivery date (3 business days from the order).
* **Account Management:** Secure registration and login with data validation for personal information.

### Administrative Capabilities
* **Inventory Management:** Full CRUD operations for products and categories.
* **Order Tracking:** Overview and management of all customer orders.
* **User Administration:** Ability to view and update registered user details.

## 🛠️ Tech Stack

* **Backend:** C# | ASP.NET Core MVC (.NET 8)
* **Database:** Microsoft SQL Server | Entity Framework Core (Code First approach)
* **Security:** ASP.NET Core Identity for authentication and role-based authorization (Admin/User)
* **Frontend:** HTML5, CSS3, and Bootstrap for a fully responsive design
